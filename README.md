# 📌 Project Overview

This project focuses on designing and implementing a scalable and efficient **Hospital Local Area Network (LAN)** infrastructure using Cisco networking concepts.

The system provides reliable communication among hospital departments while ensuring scalability, optimized resource sharing, structured IP allocation, and network efficiency.

---

# 🎯 Objectives

✔ Design a Hospital LAN Network

✔ Configure Routers and Switches

✔ Implement IP Address Allocation

✔ Enable Communication Between Subnets

✔ Perform Cost Analysis

✔ Improve Network Reliability
